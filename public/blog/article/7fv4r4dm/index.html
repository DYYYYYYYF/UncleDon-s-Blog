matplotlib基础用法及简单设置

安装并导入matplotlib库

安装：

pip install matplotlib

如果安装失败可以尝试更换镜像源。

导入(表格)：库中还有其他模块可以导入，比如animation。

import matplotlib.pyplot as plt

设置表格大小

这里的width/height单位皆为英寸（1英寸=100像素）。

 fig = plt.figure(figsize=(width, height)

设置表格边距

边距设置范围为 0-1。

plt.subplots_adjust(left=0, right=1, top=1, bottom=0)
